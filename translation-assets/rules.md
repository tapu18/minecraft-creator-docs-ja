# 翻訳ルール

- 原文の構造を維持する（見出し・箇条書き・表・コードブロック）
- コードブロックとインラインコードは変更しない。ただしコードブロック内のコメントのみに限り翻訳する。
- リンク先URLとアンカーは変更しない
    - ドキュメント間のアンカーのラベルについて、アンカー先のタイトルが使用されている場合は、アンカー先のドキュメントが未翻訳の場合でも日本語に翻訳したタイトルを使用する。その際にtranslation-assets/titles.ymlに翻訳したタイトルを追記する。
    - 見出しを翻訳する場合、翻訳後の見出し直前に `<a id="原文の英語スラッグ"></a>` を追加してアンカーIDを固定する。
- 用語は glossary.yml に従って統一する
