# minecraft-creator-docs-ja
[MicrosoftDocs/minecraft-creator](https://github.com/MicrosoftDocs/minecraft-creator) のドキュメントを翻訳するプロジェクト。

## ディレクトリ構成
- `upstream/` (submodule): 原文
- `docs-ja/`: 翻訳成果物
- `translation-assets/`: 翻訳ルール・用語集・翻訳対象定義
- `ai/`: AIエージェント用プロンプト・設定
- `scripts/`: 変換・差分・QA用スクリプト
